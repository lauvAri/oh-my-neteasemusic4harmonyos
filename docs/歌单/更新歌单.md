更新歌单

参数：

- id:歌单id
- name:歌单名字
- desc:歌单描述 限制100个字
- tags:歌单tag ,多个用 `;` 隔开,只能用官方规定标签

请求：

POST /playlist/update?id=18060504768&name=新歌单&desc=描述&tags=欧美

响应：

```json
{
    "/api/playlist/tags/update": {
        "code": 200
    },
    "/api/playlist/desc/update": {
        "code": 200
    },
    "/api/playlist/update/name": {
        "code": 200
    },
    "code": 200
}
```
