搜索建议(联想)

参数说明：

- 必选参数：
    - keywords : 关键词

请求：

POST /search/suggest?keywords=lauv

响应：

说明：该部署的 `/search/suggest` 返回的不是关键词列表，而是**多类型搜索结果**（单曲 / 歌手 / 专辑），由 `result.order` 指明分组的出现顺序。

```json
{
    "code": 200,
    "result": {
        "order": [
            "songs",
            "artists",
            "albums"
        ],
        "songs": [
            {
                "name": "Paris in the Rain",
                "id": 564000295
                // 其余字段同 /search(type=1) 的单曲结构
            },
            // ...
        ],
        "artists": [
            {
                "name": "Lauv",
                "id": 1057226
                // 其余字段同 /search(type=100) 的歌手结构
            },
            // ...
        ],
        "albums": [
            {
                "name": "I met you when I was 18. (the playlist)",
                "id": 39087647
                // 其余字段同 /search(type=10) 的专辑结构
            },
            // ...
        ]
    }
}
```

备注：客户端联想功能按 `songs → artists → albums` 的顺序提取各分组的 `name`、去重并截断，作为联想词条展示。
